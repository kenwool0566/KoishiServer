## [🚀] Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/kenwool0566/koishiserver
cd koishiserver
```

### 2️⃣ Run the Server
Ensure you have `.NET 8.0 SDK` installed:
```bash
dotnet run --project Program
```
